# План перехода: Frontend Developer -> Cybersecurity Specialist

Этот путь сфокусирован на **Application Security (AppSec)** и **Web Pentesting**, так как здесь твой опыт во фронтенде дает максимальное преимущество.

---

## Этап 1: Смена фокуса (Web Security Fundamentals)
*Изучаем, как ломается то, что ты раньше строил.*

* **OWASP Top 10:** Глубокое изучение десяти самых критичных рисков (XSS, SQLi, CSRF, IDOR и т.д.).
* **Безопасность браузера:** Разберись с Same-Origin Policy (SOP), CORS (как его обходят) и заголовками безопасности (CSP, HSTS).
* **Инструментарий:** * **Burp Suite (Community Edition):** Главный инструмент для перехвата трафика.
    * **DevTools:** Использование вкладки Network и Console для поиска утечек данных.

## Этап 2: Расширение кругозора (Infrastructure)
*Выходим за пределы браузера.*

* **Сети:** Понимание стека TCP/IP, работы DNS, TLS/SSL и модели OSI.
* **Linux:** Свободное владение Bash, управление правами (chmod/chown), работа с логами.
* **Базы данных:** Основы SQL для понимания векторов инъекций.

## Этап 3: Практика (CTF и Лаборатории)
*Легальный взлом для набивания руки.*

1.  **PortSwigger Academy:** (Бесплатно) — Лучший ресурс для веб-безопасника. Пройди все темы от XSS до бизнес-логики.
2.  **TryHackMe:** Пройди ветку "Web Fundamentals".
3.  **Hack The Box:** Площадка для более продвинутых задач, когда освоишь базу.

## Этап 4: Специализация
*Выбери, что тебе ближе:*

* **AppSec / DevSecOps:** Внедрение безопасности в процесс разработки (анализ кода, настройка сканеров в CI/CD).
* **Pentesting:** Проведение тестов на проникновение по заказу компаний.
* **Bug Bounty:** Поиск уязвимостей в реальных продуктах за вознаграждение (HackerOne, Bugcrowd).

## Этап 5: Сертификация
*Для подтверждения квалификации перед работодателем.*

* **eJPT:** Отличный старт для новичков.
* **OSCP:** Сертификат "золотого стандарта" для пентестеров.
* **CASE:** Если решишь уйти в глубокую защиту приложений.

---
**Твой главный козырь:** Знание того, как работает JS "под капотом". Используй это для анализа сложной логики на клиенте, которую системные администраторы часто упускают.
